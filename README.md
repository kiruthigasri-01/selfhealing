<h2>Self-Healing Commerce Workflow Orchestrator</h2>

<p>
This project implements an AI-powered workflow orchestration framework for ecommerce data pipelines.
It performs dataset transformations, validates business rules, investigates failures using
Retrieval-Augmented Generation (RAG), and enables automated recovery actions.
</p>
